Steps to install the dotfiles:
1. install zsh
2. install oh-my-zsh:
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
3. install tmux
4. install vim
5. install Vundle git clone https://github.com/VundleVim/Vundle.vim.git
~/.vim/bundle/Vundle.vim
6. copy .tmux.conf 
7. copy .vimrc
8. copy .zshrc
9. fix paths in .zshrc
10. run PluginInstall in vim
11. fuck around with YCM
12. install powerfonts and setup vim-airline
